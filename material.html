Material

From this page you can access the folder with all the useful material related to the Computer Engineering course.

It is also possible to upload files anonymously and without having special accounts, just upload a file and administrators will sort it in the shared cloud.

You can upload any type of file to the archive, but books, video lessons or other material explicitly protected by copyright will not be published.

Link MEGA folder

The files you find in this folder have been sorted, but there may still be errors. For any type of report, contact the administrators.

File upload link

Through this link, even without a MEGA account, you can upload all the files you deem useful. If eligible, they will be published in the official cloud.

As for the upload from mobile, although it is quite inconvenient to share files on both iOS and Android, I created a link similar to the one above, which however works via Dropbox:

Since Dropbox has many limitations, I kindly ask you to use, if possible, upload via MEGA.

The more material we have, the more useful this folder will be. Telegram is certainly a very interesting platform for quick comparison, but for finding files and for storing handouts, notes, exam topics, etc. it can be very dispersive.

You can also upload other material (not necessarily related to the courses), provided it is of common interest and comes from reliable sources. For questions or other types of requests, contact the administrators.

I thank in advance all those who take a few minutes to fill this folder with useful material to all the students of our course.

N.B.: The managers of this folder reserve the right to remove the files present at any time, as well as to delete the entire folder without notice. Sorting of files is done without any guarantee.

Managers can make mistakes, it can take a long time to fix the files, the names of the files may be inaccurate and there may be duplication, therefore we ask you to report any type of problem in the relative group.
